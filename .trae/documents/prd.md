## 1. 产品概述

面试回答拆弹桌 — 一款面向应届生和转岗求职者的高压面试答题训练工具，帮助用户在限时压力下组织结构化表达，记录卡壳问题并生成针对性强化练习列表。

- 解决求职者面对高压面试问题时答案散乱、缺乏结构、容易紧张卡壳的核心痛点
- 目标市场为校招应届生与职场转岗人群，提供高频题库 + 结构引导 + 限时训练 + 弱点复盘的完整练习闭环

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 练习者 | 无需注册，本地使用 | 全部功能：题库浏览、计时练习、复盘记录 |

### 2.2 功能模块

1. **拆弹训练场**：问题抽取、限时倒计时、结构提示面板、语音/文字作答
2. **弱点复盘台**：卡壳记录列表、强化练习生成、练习进度追踪
3. **题库弹药库**：按分类浏览问题、搜索筛选、难度标记

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 拆弹训练场 | 倒计时面板 | 支持 30s/60s/90s 三档倒计时，倒计时结束自动标记为"超时"，倒计时进度环动态显示剩余时间 |
| 拆弹训练场 | 问题卡片 | 从题库随机抽取或按分类选取面试问题，显示分类标签和难度等级 |
| 拆弹训练场 | 结构提示面板 | 根据问题分类自动推荐答题结构：STAR 法则 / 对比法 / 结论前置法，点击展开详细引导步骤 |
| 拆弹训练场 | 作答区 | 文本输入框记录答案，支持"标记卡壳"按钮标记当前回答卡住的位置 |
| 拆弹训练场 | 练习结果弹窗 | 显示作答时长、是否超时、卡壳次数、结构完整度评分，可选择加入弱点列表或重新练习 |
| 弱点复盘台 | 卡壳问题列表 | 按时间倒序展示所有标记为"卡壳"的问题，显示卡壳次数、最近练习时间 |
| 弱点复盘台 | 强化练习列表 | 系统根据卡壳频次自动生成强化练习队列，支持一键开始连续训练 |
| 弱点复盘台 | 练习进度看板 | 统计各分类的练习次数、卡壳率下降趋势、连续通关天数 |
| 题库弹药库 | 分类浏览 | 按自我介绍/项目经历/冲突处理/离职原因四大分类浏览问题 |
| 题库弹药库 | 搜索筛选 | 关键词搜索 + 分类筛选 + 难度筛选 |
| 题库弹药库 | 问题详情 | 展示问题文本、推荐答题结构、示例答案要点、难度等级 |

## 3. 核心流程

**主流程**：用户进入拆弹训练场 → 选择分类或随机抽取 → 查看结构提示 → 开启倒计时 → 作答 → 标记卡壳(可选) → 查看结果 → 卡壳问题自动进入弱点复盘台

**复盘流程**：用户进入弱点复盘台 → 查看卡壳列表 → 启动强化练习 → 重复训练直到通关 → 进度看板更新

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深灰底(#1a1a2e) + 琥珀橙(#ff6b35)作为警告/强调色 + 薄荷绿(#00e676)作为通过/完成色
- **辅助色**：冷灰(#2d2d44)卡片底色，红色(#ff1744)表示超时/卡壳
- **按钮风格**：圆角矩形，带微妙阴影和 hover 发光效果，主按钮使用琥珀橙填充
- **字体**：标题使用 Noto Sans SC Bold，正文使用 Noto Sans SC Regular，倒计时数字使用等宽字体 JetBrains Mono
- **布局风格**：左侧导航栏 + 主内容区的经典双栏布局，训练场采用中心聚焦式布局
- **图标风格**：线性图标，配合炸弹/拆弹隐喻的视觉元素

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 拆弹训练场 | 倒计时面板 | 居中大号圆形进度环，JetBrains Mono 数字，进度环颜色随时间变化(绿→橙→红)，倒计时结束闪烁动画 |
| 拆弹训练场 | 问题卡片 | 居中偏上位置，深色卡片+左侧分类色条，分类标签+难度星标 |
| 拆弹训练场 | 结构提示面板 | 右侧侧边栏，可折叠展开，STAR/对比法/结论前置步骤用步骤条展示 |
| 拆弹训练场 | 作答区 | 底部文本输入区，"标记卡壳"按钮带炸弹图标，提交按钮 |
| 弱点复盘台 | 卡壳问题列表 | 左侧列表，每项显示问题摘要+卡壳次数徽标+最近练习时间 |
| 弱点复盘台 | 强化练习列表 | 右侧面板，队列式排列，当前练习项高亮 |
| 弱点复盘台 | 练习进度看板 | 顶部卡片组，显示练习次数/卡壳率/通关天数，迷你趋势图 |
| 题库弹药库 | 分类浏览 | 四大分类卡片，每个卡片带图标+题目数量+分类色 |
| 题库弹药库 | 搜索筛选 | 顶部搜索栏+分类标签筛选+难度筛选下拉 |
| 题库弹药库 | 问题详情 | 模态弹窗，展示问题+推荐结构+示例要点 |

### 4.3 响应式设计

- 桌面优先设计，核心训练流程在大屏上获得最佳体验
- 平板端：侧边栏折叠为底部导航，结构提示面板改为底部抽屉
- 移动端：单栏布局，倒计时环缩小，问题卡片全宽
